Entering A Case

How to Enter a Case in eCourt

  1. On the left hand menu, select the Add Case/Case Initiation navigation group.

  2. Fill out any needed information, marked with: *

  3. Click Save & Open, or Save & Add Another

  4. The case will then open in your current browser tab, or create the case and navigate back to the case add form.
