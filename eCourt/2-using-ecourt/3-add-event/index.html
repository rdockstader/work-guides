Adding an Event

How to Add an Event to a Case in eCourt

Events in eCourt can be handled several different ways. They can be added to time slots, dockets, or Just to the case. The most common one is just to the case (as dockets and time slots can get a little more complicated) so we will be going over that method.

  1. From an existing case, find the add event/add scheduled event/schedule event button. This can be commonly found under an events heading, or the summary heading.

  2. Fill out any information that is needed. Required info is marked by: *

  3. Hit one of the Save buttons on the bottom of the form.

  4. The Event should then exist on the case
