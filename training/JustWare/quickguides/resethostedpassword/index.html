Resetting Hosted Password

How to reset a users password on our hosted servers

  1. Open File Explorer

  2. Type \\jove\support\Hosted\RDCMan 2.7 in the nav bar

  3. If you don't have Remote Desktop Connection Manager installed on your machine, run rdcman.msi and complete the installation with the default options

  4. Follow these steps if you haven't installed the cert:

    1. Open JTIRDCManCert pw.txt and copy the password to your clipboard

    2. Open JTIRDCManCert.pfx

    3. Leave Current user selected, and hit next

    4. The file name should be the file that you opened. Verify, and hit next

    5. Input the password from your clipboard, leave the default options and hit next

    6. Leave the default and hit next

    7. Hit finish on the last page

  5. With RDCMan and the Certificate installed, launch Hosted Shared Saas (RDCMan 2.7).rdg

  6. Select the customer agency you're resetting the password for from the server list

  7. Once connected, seletec the key ring icon on the task bar

  8. enter the Username and press the binoculars

  9. If there are multiple matches, you'll have to select the correct username from the list

  10. With the user selected, verify that the name of the user is correct and type in a new password. (the G button will generate a random password)

  11. With the new password entered verify Force Password change at Login is selected, and hit change password. If the account is locked, you can also unlock the account.

  12. The password should now be changed. Sign Out of the computer, and notify the admin of the new password for the user.
