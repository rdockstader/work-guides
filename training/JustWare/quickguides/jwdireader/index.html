Finding JWDI Files, and Using the JWDI Reader

Where to find JWDI foles and how to use the JWDI Reader

JWDI Files are the files that JustWare uses while submitting documents from the client machine to the server. When there is an error with a file upload, those errors get written to the JWDI file for that upload.

Finding JWDI Files Using the JWDI Reader

Finding the JWDI Files

  1. First, open the upload manager to check if there are any failed documents

  2. Open File explorer and type %LOCALAPPDATA% into the filepath at the top

  3. Open the NewDawnTech folder

  4. Open the JustWare5 folder

  5. Open the Folder Coresponding to the instance your troubleshooting

  6. Open the DocumentsTemp folder

  7. Open the uploads folder

  8. In the upload folder, you should see a jwdi file and the upload that failed

Using the JWDI Reader

  1. With the files located. Download the JWDI Reader from \\satools\Apps\JustWare Specific\JwdiReader.exe

  2. Run the JWDI Reader, and select Open file

  3. paste in the path to the uploads folder

  4. Select the JWDI file, and click Open

  5. This should load the information for the error message/file into the JWDI reader
