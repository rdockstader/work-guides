Removing JustWare local AppData cache

How to use the batch file to remove the local appdata cache

  1. Download this batch file

    Download

  2. Close all instances of JustWare that are open

  3. Run the batch file

  4. If you get a prompt, select more info and then run anyway

  5. Press Y on the do you want to delete apps/2.0 folder

  6. If you don't have access to the files (I.E. you need admin access, or you currently have justware open) you'll get errors like this

  7. If you don't get any errors, hit Y on the next prompt

  8. You should now be able to relaunch justware, and have it re-install.
