Add/Remove Restore Script Users

How to add and remove users from the Support Restore Script

Jump To:

Add a New User Remove a User

Add a New User

  1. Open SQL Server Management Studio (SSMS)

  2. Connect to Zombie

  3. Run the AddUsertoRestoreScript.sql (download below) script replacing the following:

    • First Name
    • Last Name
    • Hire Date
    • Role (if needed, SUP = Support Tech, ESC = Escalated Tech, SA = Solutions Architect)
    Download the Add User Script

Remove a User

  1. Open SQL Server Management Studio (SSMS)

  2. Connect to Zombie

  3. Run the RemoveUserFromRestoreScript.sql script, replacing the username with the username you are removing

    Download the Remove User Script
