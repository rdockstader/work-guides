JustWare Installation Guide

How to install JustWare against a Database

Jump To:

Check Database Version Upgrade/Downgrade the Maintance Console Create Install Folders Install JustWare

Check Database Version

  1. Open SQL Server Management Studio (SSMS), and connect to the SQL Server

  2. right click on your database, and select new query

  3. Run the Check Version SQL Query

    Download the Query

  4. The result of this query is your JustWare Version

Upgrade/Downgrade the Maintance Console

This is only necessary if the Maintance Console is not already on the same version as your database.

  1. Connect to Acuna in Remote Desktop Connection

  2. Launch the JustWare Maintance Console, and check the version

  3. If this version doesn't match your database version, close the console.

  4. Open task manager -> users, and check if any other users (active or disconnected) are using the console

  5. If an active user has it open, message them asking if they can close it so you can upgrade it. If it is a disconnected user kill the task

  6. open File explorer, and go to \\jove\Product_Development\Installations\JustWare\ and select the version that matches your database

  7. Open the MaintenanceConsoleInstaller folder, and copy the JWMCInstaller.exe to your desktop

  8. Run the JWMCInstaller.exe

  9. If it's an upgrade (increase in version number) there will be an upgrade button. If that is there, select it. Otherwise run it in Repair

  10. Select Repair/Upgrade on the next screen

  11. If you see this message the sync service needs to be killed.

  12. To do this, launch task manager, go to the services tab, find JustWare Syncronization Service, right click and select Go To Details..

  13. Right click the details row and end task

  14. Select OK on the prompt

  15. After the installer completes hit OK, open the console back up, and you'll be ready to Install JustWare

Create Install Folders

  1. On the Web server, open file explorer

  2. Open the S Drive

  3. Open JustWare Installs

  4. If there is a folder with your initials in there, open that folder. Otherwise Create one with your initials on open it

  5. Create a new folder with the Agency name (the middle part of the database name), and open that folder

  6. Create Three folders inside the agency folder

    • docTemp
    • workflowTemp
    • filingCabinet

Install JustWare

  1. Navigate to \\jove\Product_Development\Installations\JustWare\ and find the JustWare version that matches your database

  2. Open the Updates folder, and choose any of the .zip files there

  3. Copy the file to your desktop

  4. Open the JustWare Maintenance Console, and go to the Web Server Install tab

  5. Select Browse next to the Install File text box, and brose to your desktop where you placed the .zip file

  6. name the installation the same as your database, but without underscores. I.E. RDTest, or RDGuam18150

  7. Click the Install button (the next prompt may take a while to load)

  8. On the first page, select Next

  9. Open \\jove\support\License 9999 (internal use only).txt, and paste in that licence key

  10. Varify the host name matches the server your on, and hit next

  11. Select Next

  12. Fill in the SQL Server Instance (the SQL Server Name I.E. Zombie or Zeppelin) and the JustWare Database Name (the name of the database. I.E RD_Test or RD_Guam_18-100). Then hit next

  13. Change the External address to the SQL Server for both URLS. This should also update the internal address
