Install SharePoint on Windows 2016

How to install SharePoint 2016 on Windows Server 2016 for JustWare Document Searching

Ryan Dockstader | Sept 24, 2018

Install SharePoint Pre Requisits

  1. Open the .iso

  2. Run splash.hta

  3. Select Install software Prerequisits

  4. Select Next on Page 1

  5. Accept the Terms and click next on page 2

  6. Let the installer install the Pre-Reqs (This can take a while)

  7. Once the installer completes, select finish. (THIS WILL RESTART THE MACHINE)

  8. Repeat until all Pre Reqs have completed Installing

Installing Sharepoint 2016

  1. Open the .iso

  2. Run splash.hta

  3. Select Install SharePoint Server

  4. Input a Licence Key and click Continue on page 1

  5. Accept the Terms and click Continue on page 2

  6. Select the file locations (the default is fine) and select install

Configuring Sharepoint 2016

  1. Click Next on the first page

  2. Select Yes on the Prompt

  3. If this a new setup, select Create a new server farm. Then click Next

  4. fill in the database server and name information, then click next

    Note: the username and password must be a domain username and password

  5. Enter in a passphrase, and click next

  6. Select Single-Server Farm

  7. You can select a port if you would like, or just let the installer pick for you. Leave the setting as NTLM, then hit next

  8. Double check the settings, then click next

  9. Wait for the configuration to finish (this can take a while)

  10. Select Finish on the final page

SharePoint Administration Site

  1. When the Site first load, select No, I don't wish to partipate and click OK

  2. Click the Start the Wizard button on the Welcome Page

  3. Choose use existing managed account, and verify that Search Service Application is selected. Then click Next

    Note: THE OTHER BOXES CAN BE UNCHECKED

  4. Fill in the title and Description. You can set a custom URL if this a shared server (I.E. sharepoint and JustWare are running on the same server. Select the Basic Search Center template under the Enterprise heading. Select OK

  5. This should install a site to search documents outside of JustWare. On the final page, select finish to complete the setup

Change the Port for Sharepoint

This is only necessary if something else is running on port 80 (such as JustWare)

Note: This may have to be done as the local machine administrator.

  1. Select Manage web applications

  2. Select the site that was created (it should be running on port 80)

  3. Click the Delete X on the top bar

  4. Select yes on Delete Content databases and Delete IIS web sites. Then hit delete

  5. After it finishes Deleting, return to the home page and select Create site collections

  6. Select New Web Application

  7. On the New Web Application screen, change the port number to something other then 80. Leave other settings as they are. Then hit OK

  8. On the create Site collection screen, type a title and select Basic Search Center from the Enterprise tab.

  9. Scroll down and ad dyour user name as the primary site collection administrator then hit OK.
