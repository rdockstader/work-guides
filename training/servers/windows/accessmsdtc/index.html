How to Access the MSDTC

How to access the MSDTC on a Windows Server

Ryan Dockstader | Sept 20, 2018

  1. Open start and search for dcomcnfg

  2. expand Component Services -> Computers -> My Computer -> Distributed Transaction Coordinator

  3. Right click on Local DTC and click Properties

  4. Select the security tab

  5. Verify these settings are the same across all communicating server (web, database, etc.)
