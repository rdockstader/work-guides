Limiting Memory Usage on MS SQL Server

How to limit the memory uwage on a Microsoft SQL Server

  1. Open SQL Server Management Studio (SSMS)

  2. Connect to the SQL Server you want to limit memory on

  3. Right click on the Root level (Should say the database name) and select properties

  4. In the Select a Page section, select Memory

  5. Set the maximum memory usage to as much as the service should take up on the server

    Depending on what the server does, this will very. If it's just SQL then you can set it to ~2 GB under available server memory. If it's shared, you'll need to evaluate how much memory everything else running on the server will need.

    Note: The default value is basically infinity.

  6. Hit OK to save
