Installing Intellij for eCourt

How to Install, configure, and connect Intellij for eCourt

Jump To:

Installing the JDK on Windows

  1. go to the Java JDK download page

  2. Select the latest Java SE

  3. Under Java SE Development Kit, select accept licence agreement

  4. download the 64 bit windows .exe

  5. Run the .exe that downloaded

  6. Select Next on the first page

  7. Select Next on the second page

  8. The JDK should then install. When finished select Close. The JDK is now ready to go.

Installing Groovy on Windows

  1. Open the Apache Groovy Download Page and selec the download button

  2. After the download is complete, extract the .zip

  3. While the .zip is extracting, open C:\Program Files and create a new folder called Groovy

  4. Once the .zip completes extracting, open the extraced folder and move the Groovy-VERSION folder into your C:\Program Files\Groovy directory

  5. Once this is completed, Groovy is available on your machine.

Installing Intelij

  1. Go to the Intellij IDEA download page

  2. Download the Community Edition

  3. Launch the installer .exe

  4. Select next on the first page

  5. Select next on the second page

  6. select the boxes for .java and .groovy, and 64-bit launcher. Then hit next

  7. select Install on the fourth page

  8. After the installer is finished, check the box to run IntelliJ and hit finish

First Time Setup

  1. Select Do Not import settings

  2. Scroll to the end and accept the privacy policy

  3. Select Don't send

  4. Select a Theme (Darcula is the most common choice), then hit next: Default plugins

  5. Leave the defaults, and select Next: Featured plugins

  6. Leave the defaults, and select Start using IntelliJ IDEA

Configuring Project Settings

  1. On the start page of IntelliJ, select configure -> Project Defaults -> Project Structure

  2. on the Project page under Project SDK select new

  3. Select JDK from the dropdown

  4. Drill down into the instalation folder for the JDK (C:\Program Fiels\Java is the default location), select the JDK and click OK

    Note: The Project SDK should then update

  5. Update the Project language level to the same version as your SDK (I.E. I have version 11 SDK, so I set my Project Language Level to 11)

  6. Select OK

  7. Back on the IntelliJ IDEA Start page, select Configure -> Settings

  8. Expand Editor -> General -> Auto Import

  9. Check Add unambiguous imports ont he fl and optimize imports on the fly

  10. Select Appearance and make sure Show Line Numbers is checked

  11. Select Code Folding and uncheck imports, one-line methods, closure, and generic constructor and method parameters (so no code is hiden. If you're experience in Java and InteliJ IDEA you may not need to do this step.)

  12. Click OK, and you're ready to get your first project setup!

Mapping a network drive

  1. Launch File Explorer

  2. Browse to \\acdc\Tomcat Installs, and navigate to
    \your initials\install name\webapps\sustain\WEB-INF
    Copy this path.

  3. Right click on This PC and select Map Network Drive...

  4. Select a drive letter that isn't used, and paste in the path copied in the previous step into the folder. Then Hit Finish

  5. Your network drive should be now mapped, and you're ready to create your project and link to the eCourt libraries.

Creating a Groovy Project, and linking to the eCourt libraries

  1. Open Intellij, and create a new project

  2. Check the box for Groovy

  3. On use library, select "create" and drill down to the groovy SDK that was just downloaded

  4. Click Next

  5. Give your Project a Title (I recommend calling it 'eCourt Groovy Scripts' or something to that effect). Then hit finish.

  6. Open File -> Project Structure

  7. Select Libraries

  8. Press the + button, and select Java

  9. Select the drive that was mapped in Mapping a network Drive

  10. Intellij Will then scan the directory and list a module. Click OK once the scan is done.

  11. Once the Library is added, press th e+ and add the \lib folder

  12. Press the + again, and this time add the classes folder

  13. Once those are added, hit OK on the Project Structure dialog

  14. Wait for the Index to complete

  15. Once the index is complete, open Project explorer, right click on the root folder and create a new folder. Call this something like "RDTest Scripts" (where RDTest is the name of the environment you're running against.)

  16. Right click on the file, and create a new groovy script (you can name this after a business rule you're creating, just name it test script, or whatever you'd like).

  17. As Lincoln would say, you should now be ready to Groove!
